ポジション利確モニター

SBI証券の約定履歴CSVを端末内だけで読み込み、保有ポジションと利確ラインをチャート表示します。CSVはどこにも送信・保存されません。

🔒 このCSVはサーバーに一切送信されません。ブラウザ内のJavaScriptだけでファイルを読み込み・解析しています。通信が発生するのは、公開情報である株価データ（docs/prices/）の取得のみです。

保有ポジション

CSVを選択すると、ここに保有ポジション別のチャートが表示されます。
利確/損切りラインの設定はこの端末の localStorage にのみ保存されます。
株価データは J-Quants API から取得し、平日夜間に自動更新。
本ツールの表示は投資判断の参考情報であり、投資助言ではありません。
